# .taskmaster

This folder holds the **Hunter** taskmaster: a single entry point for app description, architecture, schema, APIs, deployment, and document index.

**Start here:** [TASKMASTER.md](./TASKMASTER.md)

It describes:

- What Hunter is (backend + frontend + Supabase)
- Schema (single source of truth: `supabase_schema.sql`)
- Backend commands and endpoints (webhook, /api/run)
- Frontend alignment and proxy
- Apify webhook URL and setup
- Deployment (Railway, Vercel)
- Full document index with one-line descriptions
- Quick reference (URLs, env vars)

Keep TASKMASTER.md updated when schema, APIs, or deployment change.
